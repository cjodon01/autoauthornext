## 🚀 **Quick Start**

1. **Setup Development Environment**
   ```bash
   npm install
   npm run dev
   ```

2. **Build for Production**
   ```bash
   npm run build
   npm start
   ```

3. **Deploy to Vercel**
   ```bash
   npm run deploy
   ```
